Descripción

Pequeña aplicación que permite llevar adelante partidos y resultados en un torneo de fútbol. Se pueden ingresar equipos y luego generar resultados entre ellos, donde la tabla de posiciones se actualiza automáticamente. Posee alertas de equipos repetidos y permite ingresar partidos sin resultados para asignarlos después. La tabla de posiciones se muestra según los puntos, pero se puede reordenar según alguna columna ingresada por el usuario. Desarrollado con Php y Bootstrap. La base de datos es MySql y posee las relaciones para funcionar de manera automática.
