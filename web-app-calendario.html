Descripción

App que permite ABM de cirugías para una empresa que presta servicios a las mismas. Permite el ingreso de todos los detalles de la cirugía, como fecha y hora, médicos, pacientes, servicios prestados, facturación. Realizado con Php usando una base de datos MySql y siguiendo una programación lineal, renderizando todo desde el servidor. Este fue uno de mis primeros proyectos, que me permitió conocer en detalle el funcionamiento de Php. Para probar el sistema, ingresar con usuario admin, clave 1234
