Descripción

App que permite ingresar una ciudad en un input, la cual busca en una api y devuelve las 8 coincidencias de ciudades similares, y luego pulsando en el ícono del clima se apunta a otra api la cual devuelve información sobre el estado del tiempo en esa ciudad. Este proyecto fue realizado con Express y luego reconvertido con NextJS
